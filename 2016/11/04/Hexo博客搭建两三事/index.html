github与coding的部署

部署的话，首先在github和coding（原来的gitcafe，被coding收购）官网注册。。具体过程。。。参考使用hexo搭建github.io博客(一)。总之，部署的代码部分很简单： 

hexo clean //删除source与data，最好每次部署都运行，否则标签、分类数会出错
hexo generate //可以用hexo g代替
hexo deploy //可以用hexo d代替

然后，因为我刚开始是部署在github上的，但是，由于莫名的原因，显示不完整，内容无法显示，查阅资料说原因是墙。。索性在github作为备份，在coding作为主要托管的地方。在部署的方面，与官网上的对应就好。。。因为是同时部署在两个地方，贴一点代码作为参考： 

deploy: 
  type: git
  repo: 
        github: git@github.com:yzhao94/yzhao94.github.io.git,master
        gitcafe: git@git.coding.net:yzhao94/yzhao94.git,master

在根目录配置里输入即可。
总之一句话，配置很简单。。部署各种问题。。。。（不知道是不是我的问题），总之参考上面的文章解决，还无法解决，只能左转度娘了。。。
